Randoku

hello, this service generates random numbers, so you don't have to.

API

This service was made to be queried from command-line tools (note that the / endpoint displays the API overview page if the user agent starts with Mozilla/5.0)

GET / - generate a random number between 0 and 100
/<to> - generate a random number between 0 and to
/<from>/<to> - generate a random number between from and to
/shuffle/<items> - shuffles items (a comma-separated list)
Numbers passed as parameters need to be between 0 and 18446744073709551615.
